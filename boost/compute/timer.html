Description

The timer class represents a timer.

For example, to measure the time it takes to transfer a vector of data from the host to the device:

timer public construct/copy/destruct

  1. timer(const command_queue & queue);

    Creates a new timer for queue. 

  2. timer(const timer<T> & other);

    Creates a new timer object as a copy of other. 

  3. timer& operator=(const timer<T> & other);

    Copies the timer from other. 

  4. ~timer();

    Destroys the timer object.

timer public member functions

  1. void start();

    Starts the timer. 

  2. void stop();

    Stops the timer. 

  3. bool is_stopped() const;

    Returns true if the timer is stopped. 

  4. ulong_ elapsed() const;
