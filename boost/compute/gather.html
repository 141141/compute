Function template gather

boost::compute::gather

Synopsis

// In header: <boost/compute/algorithm/gather.hpp>


template<typename InputIterator, typename MapIterator, 
         typename OutputIterator> 
  void gather(InputIterator first, InputIterator last, MapIterator map, 
              OutputIterator result, command_queue & queue);

Description

Copies the elements from the range [first, last) to the range beginning at result using the input indices from the range beginning at map.
