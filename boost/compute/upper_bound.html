Function template upper_bound

boost::compute::upper_bound

Synopsis

// In header: <boost/compute/algorithm/upper_bound.hpp>


template<typename InputIterator, typename T> 
  InputIterator 
  upper_bound(InputIterator first, InputIterator last, const T& value, 
              command_queue & queue);

Description

Returns an iterator pointing to the first element in the range sorted range [first, last) that is not less or equal to value.
