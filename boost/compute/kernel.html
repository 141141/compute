Description

Once created, kernels are executed on a compute device using one of the following methods:

  • command_queue::enqueue_nd_range_kernel()
  • command_queue::enqueue_task()

See also: program, command_queue

kernel public construct/copy/destruct

  1. kernel(const program& program, const std::string& name);

    Creates a new kernel object from program with name. 

  2. explicit kernel(cl_kernel kernel, bool retain = true);

    Creates a new kernel object for kernel. If retain is true (the default) the reference count for kernel will be incremented. 

  3. kernel(const kernel & other);

    Creates a new kernel object as a copy of other. 

  4. kernel(kernel && other);

    Move-constructs a new kernel object from other. 

  5. kernel& operator=(const kernel & other);

    Copies the kernel from other. 

  6. kernel& operator=(kernel && other);

    Transfers the ownership of the kernel from other. 

  7. ~kernel();

    Destroys the kernel object.

kernel public member functions

  1. std::string name() const;

    Returns the name of the kernel. 

  2. size_t arity() const;

    Returns the number of arguments that the kernel function expects. 

  3. program get_program() const;

    Returns the program for the kernel. 

  4. context get_context() const;

    Returns the context for the kernel. 

  5. template<typename T> T get_info(cl_kernel_info info) const;

    Returns information about the kernel.

    For example, to get the name of the kernel (without using the name() method): 

    std::string function_name = kernel.get_info<std::string>(CL_KERNEL_FUNCTION_NAME);


  6. void set_arg(size_t index, size_t size, const void* value);

    Sets the argument at index to value with size. 

  7. template<typename T> void set_arg(size_t index, const T& value);

    Sets the argument at index to value. 

  8.  operator cl_kernel() const;

    Converts the kernel object to a cl_kernel.

    This allows for kernel objects to be passed directly to functions expecting a cl_kernel argument: 

    boost::compute::kernel kernel = ...

cl_uint arity;
clGetKernelInfo(kernel,
                CL_KERNEL_NUM_ARGS,
                sizeof(cl_uint),
                &arity,
                0);
std::cout << "arity = " << arity << std::endl;
