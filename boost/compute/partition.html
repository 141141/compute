Function template partition

boost::compute::partition

Synopsis

// In header: <boost/compute/algorithm/partition.hpp>


template<typename Iterator, typename UnaryPredicate> 
  Iterator partition(Iterator first, Iterator last, UnaryPredicate predicate, 
                     command_queue & queue);

Description

Partitions the values in the range [first, last) according to predicate.

See also: is_partitioned()

Returns:

 An iterator pointing to the end of the true values.
