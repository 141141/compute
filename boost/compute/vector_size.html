Struct template vector_size

boost::compute::vector_size

Synopsis

// In header: <boost/compute/type_traits/vector_size.hpp>

template<typename T> 
struct vector_size {
};

Description

The vector_size meta-function returns the size (number of components) of the vector type T. For scalar types the function returns 1.

Examples:

  • vector_size<float>::value == 1
  • vector_size<float4_>::value == 4
