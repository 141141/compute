Struct template scalar_type

boost::compute::scalar_type

Synopsis

// In header: <boost/compute/type_traits/scalar_type.hpp>

template<typename T> 
struct scalar_type {
};

Description

The scalar_type meta-function returns the scalar type for a vector type T.

Examples:

  • scalar_type<int2_>::type == int
  • scalar_type<float4_>::value == float
