Function template remove_if

boost::compute::remove_if

Synopsis

// In header: <boost/compute/algorithm/remove_if.hpp>


template<typename Iterator, typename UnaryPredicate> 
  Iterator remove_if(Iterator first, Iterator last, UnaryPredicate predicate, 
                     command_queue & queue);

Description

Removes each element for which predicate returns true in the range [first, last).
