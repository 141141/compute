Class template vector

boost::compute::vector — A resizable container

Synopsis

// In header: <boost/compute/container/vector.hpp>

template<typename T,   // value type
         typename Alloc = allocator<T>   // allocator type
         > 
class vector {
};

Description

Note: Unlike std::vector, boost::compute::vector does not initialize values when the created or resized. If initialization is needed you must explicitly call fill() on the newly allocated values.
